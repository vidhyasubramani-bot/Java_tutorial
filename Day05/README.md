### 2. Armstrong Number

**Problem**
Check whether a number is an Armstrong number.

**Example**
Input:

```
153
```

Output:

```
Armstrong Number
```

---

### 3. Automorphic Number

**Problem**
Check whether the square of a number ends with the same digits as the original number.

**Example**
Input:

```
25
```

Output:

```
Automorphic Number
```

---

### 4. Strong Number

**Problem**
Check whether the sum of the factorials of each digit equals the original number.

**Example**
Input:

```
145
```

Output:

```
Strong Number
```

---

### 5. Largest Digit

**Problem**
Find the largest digit in a given number.

**Example**
Input:

```
59372
```

Output:

```
Largest Digit = 9
```

---

### 6. Smallest Digit

**Problem**
Find the smallest digit in a given number.

**Example**
Input:

```
59372
```

Output:

```
Smallest Digit = 2
```

---

### 7. Sum of Even and Odd Digits

**Problem**
Find the sum of even digits and odd digits separately.

**Example**
Input:

```
123456
```

Output:

```
Even Sum = 12
Odd Sum = 9
```

---

### 8. Sum of First and Last Digit

**Problem**
Find the sum of the first and last digit of a number.

**Example**
Input:

```
45678
```

Output:

```
Sum = 12
```
