## 📖 Java String Problems

### 🔤 Anagram String

Two strings are called **anagrams** if they contain the same characters with the same frequency, but the characters can be arranged in a different order.

#### Examples
- `listen` ↔ `silent` ✅
- `heart` ↔ `earth` ✅
- `java` ↔ `python` ❌

#### Logic
- Check whether both strings have the same length.
- Convert them to lowercase.
- Sort both strings.
- Compare the sorted strings.
- If both are equal, the strings are anagrams.

---

### 📏 Find the Longest Word in a String

The goal is to find the longest word from a given sentence.

#### Example
**Input**
```
Java programming is interesting
```

**Output**
```
programming
```

#### Logic
- Split the sentence into words.
- Compare the length of each word.
- Store the longest word.
- Print the longest word.
